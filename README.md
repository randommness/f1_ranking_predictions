# F1 Ranking Predictions assignment
## Context:
This is an academic project on Formula 1 Ranking Predictions, with a strong focus on Data Wrangling. The project aims to formulate a machine learning model that is able to predict an F1 driver's ranking, given his driving history, constructors' history, time of race, as well as the qualifying performance.

## File(s):
- F1 Prediction Notebook.ipynb holds the code for data wrangling and machine learning model initialisation. The report also resides in this file.

## Folder(s):
- f1db_csv_mod holds all the CSV files utilized in the .ipynb file, as well as the data dictionary.

## Note(s):
All documentation, such as steps taken and reasoning, is in part 2 of the .ipynb file.
